# 📋 Как скопировать дизайн из Figma Make

Этот сайт полностью готов к копированию и вставке. Следуйте этим простым шагам.

---

## ✨ Почему дизайн копируется идеально?

✅ **Нет базы данных** - всё работает на фронтенде  
✅ **Нет импортов figma:asset** - все изображения заменены на текст/SVG  
✅ **Все шрифты из Google Fonts** - автоматически загружаются  
✅ **Чистый код** - только React + Tailwind CSS  
✅ **Нет внешних зависимостей от API** - кроме Google Maps (стандартный iframe)

---

## 🎯 Инструкция по копированию

### Шаг 1: Выделите весь дизайн в Figma Make
1. Откройте ваш проект в Figma Make
2. Выделите весь макет (или нужную секцию)
3. Нажмите **Cmd/Ctrl + C** для копирования

### Шаг 2: Вставьте в ваш проект
1. Откройте ваш Figma файл или другой инструмент
2. Нажмите **Cmd/Ctrl + V** для вставки
3. Готово! Дизайн должен скопироваться полностью

---

## 🔍 Что копируется?

### ✅ Копируется всё:
- **Структура страницы** - все секции в правильном порядке
- **Стили** - цвета, шрифты, отступы
- **Тексты** - весь контент на двух языках
- **Анимации** - плавные переходы (Motion/Framer Motion)
- **Логотип** - текстовый логотип "PERLE"
- **Иконки** - из библиотеки Lucide React
- **Карта** - Google Maps iframe

### ❌ Не копируется (это нормально):
- **Backend код** - он не нужен, так как не используется
- **Node modules** - стандартные зависимости React
- **Конфигурационные файлы** - Vite, PostCSS и т.д.

## 🛠️ Если что-то не скопировалось

### Проблема: Пустые секции
**Причина:** Возможно, не загрузились шрифты  
**Решение:** Убедитесь, что `/src/styles/fonts.css` содержит:
```css
@import url('https://fonts.googleapis.com/css2?family=Bebas+Neue&family=Inter:wght@300;400;500;600;700&display=swap');
```

### Проблема: Нет логотипа
**Причина:** Логотип теперь текстовый (не изображение)  
**Решение:** Это нормально! Логотип "PERLE" отображается текстом в шрифте Bebas Neue

### Проблема: Нет стилей
**Причина:** Не скопировались CSS файлы  
**Решение:** Убедитесь, что файлы в `/src/styles/` присутствуют:
- `fonts.css`
- `tailwind.css`
- `theme.css`
- `index.css`

### Проблема: Кнопка бронирования не работает
**Причина:** URL системы бронирования не настроен  
**Решение:** Откройте `/src/app/components/Booking.tsx` и замените URL (строки 14 и 26)

## 🎨 После копирования

### Обязательно сделайте:

1. **Настройте систему бронирования**
   - Откройте `/src/app/components/Booking.tsx`
   - Замените `https://your-booking-system.com` на вашу ссылку

2. **Обновите контакты**
   - Откройте `/src/app/components/Footer.tsx`
   - Измените email, телефон, Instagram, адрес

3. **Проверьте карту**
   - Откройте `/src/app/components/Location.tsx`
   - Убедитесь, что Google Maps показывает ваш адрес

4. **Проверьте услуги и цены**
   - Откройте `/src/app/components/Services.tsx`
   - Обновите список услуг и цены

### Рекомендуется:

5. **Протестируйте на мобильных**
   - Откройте на телефоне
   - Проверьте, что всё отображается корректно

6. **Проверьте обе языковые версии**
   - Переключитесь на немецкий
   - Убедитесь, что все тексты корректны
