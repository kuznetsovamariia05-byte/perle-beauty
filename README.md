## ⚡ Быстрая настройка (3 шага)

### 1️⃣ Система бронирования
Откройте `/src/app/components/Booking.tsx` (строки 14 и 26):
```tsx
bookingUrl: 'https://calendly.com/your-username' // ← Ваша ссылка здесь
```

## 📱 Структура сайта

1. **Hero** - Главная секция с логотипом и призывом к действию
2. **Services** - Список услуг (маникюр, брови, ресницы)
3. **Booking** - Секция бронирования с переадресацией на внешнюю систему
4. **About** - Информация о салоне и преимуществах
5. **Location** - Карта и адрес салона
6. **Footer** - Контакты, часы работы, соцсети

---

## 🛠️ Технологии

- **React 18** - UI библиотека
- **Tailwind CSS v4** - Utility-first CSS фреймворк
- **Motion** - Библиотека анимаций (Framer Motion)
- **Lucide React** - Иконки
- **Vite** - Сборщик
